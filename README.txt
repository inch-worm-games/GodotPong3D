# GodotPong3D

## Overview
GodotPong3D is an open-source implementation of the classic game Pong using the Godot Engine by Inchworm Games.

## Setup Instructions
To get GodotPong3D running on your local machine, follow these steps:

### Prerequisites
- Godot 4.2x

### Running the Game
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/GodotPong3D.git


Open Godot Engine and import the project.
Navigate to the project directory and open the project.godot file.

Press space bar to spawn a ball

### Script Descriptions ###
Ball.gd
Handles the behavior of the pong ball including movement, collision responses, and scoring.

Paddle.gd
Controls the paddle's movement using player input for the player-controlled paddle and AI logic for the computer-controlled paddle.

GameManager.gd
Manages game states, including tracking scores, serving the ball, and determining when the game ends.

UIController.gd
Manages the game UI, updating score displays, and showing messages like "Game Over" or "You Win".
